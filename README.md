# Internship Project

This repository contains Jupyter notebooks for analysis and forecasting of system resource usage.

## Contents

*   **[rp2.ipynb](rp2.ipynb)**: A notebook for analyzing CPU and Memory usage, training forecasting models (Holt-Winters, Bi-LSTM, Adaptive Ensemble), and detecting system overloads.
    *   See [RP2_README.md](RP2_README.md) for detailed documentation on this notebook.

*   **raproject.ipynb**: Core structure notebook for the project.
*   **rawithbitbraindata.ipynb**: Notebook utilizing the BitBrain dataset for analysis.

## Setup

Ensure you have the required dependencies installed (pandas, numpy, matplotlib, tensorflow, scikit-learn).
You may need to adjust file paths in the notebooks to point to your local data.
